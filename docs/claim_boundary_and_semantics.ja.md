# Claim Boundary and Semantics

本リポジトリは、Layer 0 主張を、現代的大規模言語モデルという明示定義域上の数学的機能必要条件定理として扱う。

これは全デプロイ済み・全未来モデルの経験調査ではない。それは弱点ではない。数学的定理は、定義域・公理・推論規則の上で成立する。「定義依存だ」という指摘だけでは反論にならない。それは形式定理であることを確認しているに過ぎない。

## 中核境界

| Item | Position |
|---|---|
| Domain | 現代技術用語としての大規模言語モデル |
| Formal claim | 6つの Layer 0 役割が、明示的または機能同値的に必要 |
| Executable certificate | 有限義務グラフを全列挙で証明 |
| External support | 公式provider用例と公開model-family witness |
| Valid refutation | LLMでありながら、Layer 0役割の一つと全同値物を欠くシステム |

## 非主張

本定理は、意識・意味・主体性・人間同等推論・closed-weight内部実装の経験検査を扱わない。
