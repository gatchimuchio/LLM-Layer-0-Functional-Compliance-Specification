# Objection Routing Protocol

| Objection | Routing | Response |
|---|---|---|
| “This is definition-dependent.” | theorem-domain objection | Not a refutation. Mathematical theorems are definition- and axiom-dependent. Provide a superior domain axiom or a valid counterexample. |
| “This is not empirical inspection of all LLMs.” | category error | Correct but irrelevant. This is a formal theorem, not a survey. Empirical witness mapping is Layer B. |
| “The LLM definition is private opinion.” | rebuttable | Compare with official provider usage and model-family witnesses. |
| “The candidate is an LLM but lacks Layer 0.” | valid route if demonstrated | Provide the system, ordinary technical LLM classification evidence, missing role, and absence of every functional equivalent. |
| “Future architecture may differ.” | implementation-branch objection | Accepted only if the future system remains an LLM while lacking a Layer-0 equivalent. |
