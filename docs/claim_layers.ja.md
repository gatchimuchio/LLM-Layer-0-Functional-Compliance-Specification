# Claim Layers

| Layer | 主張種別 | 支持 | Status label |
|---|---|---|---|
| Layer 0 | 用語展開による機能的必要条件仕様 | 通常技術用法 + 役割分離論証 | `FORMAL_MATHEMATICAL_FUNCTIONAL_NECESSITY_THEOREM` |
| Layer A | 宣言済み有限義務グラフの全列挙 | 64部分集合の決定論的検証 | `PROVEN_BY_EXHAUSTIVE_ENUMERATION_OF_DECLARED_OBLIGATION_GRAPH` |
| Layer B | 公開LLM系譜のwitness mapping | 公式参照 + 公開アーキテクチャ記述 | `SUPPORTED_BY_PUBLIC_WITNESSES` |
| Layer 1+ | 実装分岐分析 | Transformer、Dense、MoE、SSM、recurrent、diffusion-style、hybrid mapping | `IMPLEMENTATION_BRANCH_MAPPING` |

## 過剰主張禁止

Layer A が検証するのは宣言済み有限グラフである。Layer B が支えるのは公開world mappingである。どちらも既存・未来すべてのデプロイ済みLLMを経験的に検査したものとして記述しない。
