# Methodology

## Method stack

1. Term-explication: decompose the ordinary contemporary technical meaning of Large Language Model.
2. Role separation: preserve distinct responsibility boundaries.
3. Declared obligation graph: specify which roles are required for which obligations.
4. Exhaustive finite enumeration: check all 64 subsets of the six roles.
5. Public witness mapping: map known LLM families into the Layer-0 roles using public sources.

## Boundary

The executable appendix validates the declared finite graph. It is intentionally not presented as empirical inspection of all deployed LLMs.

## Validity condition

A successful counterexample must be an LLM in contemporary technical usage and must lack at least one Layer-0 role plus every functional equivalent.
